# Action Required: Pain Clinic Directory

Manual steps that must be completed by a human. These cannot be automated.

## Before Implementation

- [ ] **Create Mapbox account** - Sign up at https://www.mapbox.com/ to get API access (free tier: 50,000 map loads/month)
- [ ] **Generate Mapbox access token** - Go to https://account.mapbox.com/access-tokens/ and create a public token
- [ ] **Add token to environment** - Add `NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN=pk.your_token` to `.env.local`

## During Implementation

None required.

## After Implementation

- [ ] **Test geolocation permission flow** - Manually test in browser with location permissions enabled and denied
- [ ] **Verify Mapbox billing** - Check Mapbox dashboard to ensure usage is within free tier limits
