# SEO Audit — PainClinics.com

## Setup

1. Export CSVs from Google Search Console (16 months, Web only):
   - **Queries**: Performance → Search Results → Queries tab → Export
   - **Pages**: Performance → Search Results → Pages tab → Export
   - **Coverage**: Pages (Indexing → Pages) → Export

2. Place files in this directory (`docs/seo-audit/`).

3. Install Python dependencies (if not already):
   ```bash
   pip install pandas jupyter matplotlib seaborn
   ```

4. Run the notebook:
   ```bash
   jupyter lab docs/seo-audit/audit.ipynb
   ```

## Input Files

| File | Source |
|------|--------|
| `*Queries.csv` | GSC Performance → Queries |
| `*Pages.csv` | GSC Performance → Pages |
| `*Coverage*.csv` | GSC Indexing → Pages |
| `sitemaps.jpg` | Screenshot of sitemap status |

## Output Files

Generated by the notebook into this directory:

| File | Contents |
|------|----------|
| `query-clusters.csv` | Every query tagged with cluster + metrics |
| `page-performance.csv` | Every page with type, avg position, CTR, impressions |
| `quick-wins.csv` | Queries at position 4-15 with >50 impressions |
| `content-gaps.csv` | High-impression queries with no matching dedicated page |
| `action-items.csv` | Prioritized fix list |
| `index-gaps.csv` | URLs in sitemap but not indexed |
